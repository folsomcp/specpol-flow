# Installation

SpecpolFlow is not available on pip yet (coming very soon!). 

We therefore recommend to clone the specpolFlow repository in a location that is available in your $PYTHONPATH. You will need to do the same for LSDpy package https://github.com/folsomcp/LSDpy



## Pre-requisite

The SpecpolFlow tutorials make use of the following packages. We recommend installing these before using SpecpolFlow. 

* [NumPy](https://numpy.org/doc/stable/#)
* [Matplotlib](https://matplotlib.org/) 
* [pandas](https://pandas.pydata.org/)
